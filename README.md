## About

This repository contains the HA OpenCode add-on for Home Assistant.

## Installation

1. Click the button below to add this repository to your Home Assistant instance:

   [![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.][repo-btn]][repo-add]

   Or manually add the repository URL:
   ```
   https://github.com/magnusoverli/ha_opencode
   ```

2. Find "HA OpenCode" in the add-on store and click Install.
